## Introduction
Various neural network operations used in day-to-day signal processing and Artificial Intelligence (AI) applications can be represented using an Activation Function Unit (AFU), such as the sigmoid activation function. An AFU is a nonlinear computational block that maps input data to a bounded output range and plays a significant role in introducing nonlinearity in neural networks.

The AFU operates by processing input data through different functional regions based on the magnitude and sign of the input signal. Depending on the operating region, the corresponding activation output is generated using predefined logic and approximation methods. 

An AFU is defined by its input representation, approximation technique, operating regions, and the control logic used for selecting the computation path. In this project, the sigmoid activation function is divided into multiple regions such as saturation region, transition region, and linear approximation region to simplify hardware implementation and improve computational efficiency.

This project demonstrates the design and implementation of a Sigmoid Activation Function Unit using Verilog HDL. A Verilog testbench is also developed to verify the functionality and correctness of the AFU for different input conditions.
